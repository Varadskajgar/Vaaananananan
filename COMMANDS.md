# Tournament Bot Commands

## Owner Commands

### Tournament Management
- `!ct <time> <mode>` - Create tournament
- `!edit <id> <field> <value>` - Edit tournament
- `!delete <id>` - Delete tournament
- `!lock <id>` - Lock tournament
- `!unlock <id>` - Unlock tournament
- `!reschedule <id> <new_time>` - Reschedule

### Player Management
- `!players <id>` - List players
- `!cf @user <id>` - Confirm join
- `!dn @user <id>` - Confirm payment
- `!sendall <id> <msg>` - DM all players
- `!notifyall <id>` - Send reminders

### Results & Admin
- `!r <id> <text>` - Upload result
- `!status <id>` - Show status
- `!audit <id>` - Show audit log
- `!announce <id>` - Repost announcement

### Data & Exports
- `!backup` - Export backup
- `!exportcsv <id>` - Export CSV
- `!broadcast <msg>` - DM all users

## Player Commands

### Tournaments
- `!tournament` - Show today's tournaments
- `!join <id>` - Join tournament
- `!leave <id>` - Leave tournament
- `!mytournaments` - Show my tournaments
- `!info <id>` - Tournament info

### Profile & Stats
- `!stats [@user]` - Show stats
- `!profile [@user]` - View profile
- `!topplayers` - Leaderboard
- `!achievements` - View badges

### Engagement
- `!referral` - Get referral code
- `!redeem <code>` - Redeem code
- `!history` - Past tournaments
- `!feedback` - Send feedback
- `!support` - Get help
